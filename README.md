This project is a web application named "Binary Tree" designed to track carbon reduction efforts of users. The application includes multiple pages such as Index, Login, Signup, Code Input, Ranking, and Profile pages.

## Index

The Index page is the landing page of the application. It features the Binary Tree logo and navigation options.

<img width="1235" alt="image" src="https://github.com/skkuse/2024spring_41class_team4/assets/59384684/395b9e93-9c7b-489e-8099-566ccb0906a0">


**Features:**
- Navigation bar with links to Contact, Github, Login, and Signup
- Central logo with options to Start or view Ranking

## Ranking

The Ranking page displays the carbon reduction ranking of users. 

<img width="1166" alt="image" src="https://github.com/skkuse/2024spring_41class_team4/assets/59384684/2a81cc7e-cd03-4f89-8498-e16fd1041077">

- Search bar to filter by ID or Level
- Table displaying Rank, ID, and Carbon Reduction
- Buttons to navigate to user's profile

## Profile

The Profile page displays the user's profile information.

<img width="1209" alt="image" src="https://github.com/skkuse/2024spring_41class_team4/assets/59384684/677d5ed0-7eff-4ac9-a3e5-fa1241306276">


**Features:**
- Display of user ID, Email, Level, and Total Emission Reduction
- Profile image that changes according to the user's level
- Logout button
- Links to Contact and Github

## Getting Started

To get started with this project, clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/skkuse/2024spring_41class_team4.git
cd 2024spring_41class_team4/Frontend
open index.html
```
